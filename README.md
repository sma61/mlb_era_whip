# ⚾ MLB Pitcher Performance Predictor Project

An AutoML pipeline that predicts pitcher ERA and WHIP using Baseball Savant Statcast data, validates predictions against actual results, and identifies pitchers who significantly over or underperform their underlying metrics.

---

## Overview

This project trains regression models on six years of historical pitching data (2019–2024) and generates forward-looking ERA and WHIP projections. Rather than manually selecting a model, it uses **PyCaret AutoML** to systematically evaluate 20+ algorithms and select the best performer based on Mean Absolute Error (MAE) across cross-validated folds.

The 2025 season serves as a live validation set — predictions are compared directly to actual 2025 results to measure real-world model accuracy.

---

## Key Features

- **AutoML model selection** via PyCaret — evaluates Linear Regression, Ridge, Lasso, ElasticNet, Random Forest, Gradient Boosting, XGBoost, LightGBM, CatBoost, and more
- **Dual target modeling** — separate optimized models for ERA and WHIP
- **2025 validation** — predictions compared to actual results with delta analysis
- **Over/underperformer identification** — surfaces pitchers diverging most from expected performance
- **Feature importance analysis** — reveals which Statcast metrics are most predictive
- **CSV exports** — ranked projections and model comparison tables saved for further analysis

---

## Statcast Features Used

| Feature | Description |
|---------|-------------|
| `xera` | Expected ERA based on quality of contact |
| `xwoba` | Expected weighted on-base average |
| `xba` | Expected batting average against |
| `exit_velocity_avg` | Average exit velocity allowed |
| `barrel_batted_rate` | Barrel rate allowed |
| `hard_hit_percent` | Hard hit rate allowed |
| `k_percent` | Strikeout percentage |
| `bb_percent` | Walk percentage |
| `launch_angle_avg` | Average launch angle allowed |
| `p_strikeout` | Total strikeouts |
| `p_walk` | Total walks |
| `p_home_run` | Total home runs allowed |

> **WHIP methodology note:** The Baseball Savant dataset does not include hits allowed as a standalone column. WHIP is calculated as `(Walks + Earned Runs) / Innings Pitched` as a proxy. This approximation performs well given the strong correlation between earned runs and hits allowed in the dataset.

### Confidence Interval Calibration Analysis

**95% CI Coverage: 48.8%** — significantly below the 95% target.

This gap reveals an important finding: the conformal intervals were calibrated on residuals from the training period (2019–2024), where the model fits tightly. When applied to true out-of-sample 2025 data, the intervals are too narrow — the model's training-period residuals underestimate real-world prediction uncertainty.

**What the coverage gap tells us:**
- The model's point predictions are directionally correct (10 of 20 top projections landed within the CI)
- Elite, high-usage relievers (Chapman, Díaz, Muñoz, Adam) are well-calibrated — the model is most reliable for pitchers with stable, high-volume underlying metrics
- Pitchers who significantly underperformed predictions (Gage, Bautista, Stewart) may be regression candidates — their Statcast metrics suggested better outcomes than they achieved

**Planned fix:** Implement strict time-based train/validate splits (train 2019–2022, calibrate intervals on 2023–2024 holdout) to produce intervals that reflect true out-of-sample uncertainty rather than training-period residuals. See Roadmap below.

> **Notable:** Chapman, Díaz, Muñoz, and Adam all landed within 0.13 ERA of their projection — strong validation of the model's accuracy for established elite relievers. The coverage gap is concentrated among pitchers with small sample sizes or high variance profiles.

## Running the Model

```bash
python pitcher_predictor.py
```

**What happens when you run it:**
1. Loads and cleans historical pitching data (2019–2024)
2. Compares 18 regression models for ERA and WHIP via PyCaret
3. Selects and finalizes the best model for each target
4. Loads 2025 data and generates predictions
5. Compares predictions to actual 2025 results
6. Outputs top performers, over/underperformers, and saves all results to CSV

## Interpreting Results

**Delta column:**
- **Negative delta** — pitcher outperformed their projection (actual ERA/WHIP better than predicted)
- **Positive delta** — pitcher underperformed their projection (actual ERA/WHIP worse than predicted)

**Confidence interval columns:**
- **CI_Lower / CI_Upper** — the 95% confidence bounds around each point prediction
- **CI_Width** — interval width (±0.3959 for ERA). Uniform width because split conformal prediction uses a single margin derived from calibration residuals
- **In_CI** — whether the actual 2025 value fell within the predicted interval. Target: 95% coverage

**Over/underperformer analysis** identifies pitchers whose results diverge most from what their underlying Statcast metrics would suggest — a useful signal for identifying regression candidates and breakout pitchers.
